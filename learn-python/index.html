Описание

Курс "Python для начинающих" предназначен для тех, кто только начинает свой путь в программировании и хочет освоить один из самых популярных языков программирования - Python. В этом курсе вы познакомитесь с основами программирования на Python и научитесь создавать простые программы. Независимо от того, имеете ли вы предыдущий опыт программирования или нет, этот курс будет полезен и понятен для вас.
